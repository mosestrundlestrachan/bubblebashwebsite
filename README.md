# Bubble Bash Guild Website - Quick Start Guide

This is a static website for the Bubble Bash Guild. You don't need to install any heavy dependencies like Node.js to run it. You can simply use Python, which is pre-installed on macOS.

## How to Run Locally

1.  **Open Terminal** in this folder.
2.  **Run the following command**:

    ```bash
    python3 -m http.server 8080
    ```

3.  **Open your browser** and visit:
    [http://localhost:8080](http://localhost:8080)

## That's it!
You should see the website running. To stop the server, go back to the terminal and press `Ctrl + C`.
